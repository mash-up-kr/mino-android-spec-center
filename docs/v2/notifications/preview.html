MASC Discord 알림 — 상황별 메시지 미리보기

웹훅 MASC 알림이 보내는 실제 payload를 그대로 재현했습니다. 역할 멘션은 embed 밖 본문(content)에 들어가 실제로 핑이 울리고, 제목을 클릭하면 대시보드의 해당 feature로 이동합니다.

컨펌 요청

개발자가 spec을 올리고 컨펌 요청을 누르면 — spec_draft → spec_in_review. 디자이너 역할이 핑됩니다.
